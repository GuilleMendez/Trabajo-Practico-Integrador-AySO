Trabajo Práctico – Virtualización con VirtualBox y Ubuntu

Link de video de youtube: https://youtu.be/PF5I1r5rHZ4

Autores:

•	Mendez Guillermo

•	Di Gianantonio Reyes Stefano

Este repositorio contiene el desarrollo de un trabajo práctico integrador de la materia Arquitectura y Sistemas Operativos, cuyo objetivo es comprender y aplicar los conceptos fundamentales de la virtualización mediante la creación y configuración de una máquina virtual utilizando Oracle VirtualBox.
Dentro del entorno virtualizado se instaló Ubuntu Linux y se ejecutó un programa simple en Python, validando el funcionamiento del sistema operativo invitado y la interacción con la terminal.

Contenido del proyecto:

•	Documento del trabajo práctico con:
o	Introducción a la virtualización
o	Objetivos del trabajo
o	Caso práctico con VirtualBox
o	Instalación y configuración de Ubuntu
o	Ejecución de un script en Python
o	Resultados y conclusiones
•	Código de ejemplo en Python utilizado dentro de la máquina virtual
•	Capturas de pantalla de procesos de instalación, configuración, y creación del código.

Tecnologías utilizadas:

•	VirtualBox
•	Ubuntu Linux
•	Python
